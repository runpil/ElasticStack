# Query DSL의 주요 쿼리

## 1. Match All Query
- 모든 문서를 검색하는 쿼리
```json
POST movie_search/_search
{
    "query": {
        "match_all": {}
    }
}
```
## 2. Match Query
- 텍스트, 숫자, 날짜 등이 포함된 문장을 형태소 분석을 통해 텀으로 분리한 후 이 텀들을 이용해 검색 질의를 수행
- 검색어가 분석돼야 할 경우 사용
```json
POST movie_search/_search
{
    "query": {
        "match": {
            "movieNm": "그대 장미"
        }
    }
}
```
## 3. Multi Match Query
- 단일 필드가 아닌 여러 개의 필드를 대상으로 검색해야할 때 사용하는 쿼리
```json
POST movie_search/search
{
    "query": {
        "multi_match": {
            "query": "가족",
            "fields": ["movieNm", "movieNmEn"]
        }
    }
}
```
